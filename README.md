# Neck Protocol

A personal neck-posture rehab tracker: a phased release/activate/strengthen/maintain program, exercise videos, desk setup checklist, and expected timeline.

Live site: https://HamadAlsoqaih.github.io/personal-neck-protocol/

## Development

```
npm install
npm run dev
```

## Build

```
npm run build
npm run preview
```

Deployment to GitHub Pages happens automatically via GitHub Actions on every push to `main` (see `.github/workflows/deploy.yml`).
